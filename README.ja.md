## チングダチとは？

日本と韓国の間で旅行需要が爆発的に増加していますが、**言語の壁**が最大の障壁です。

チングダチは**AIリアルタイム翻訳**で韓国語と日本語の壁を取り払い、両国のユーザーが**母国語で自然に会話**できるチャットベースのネットワーキングプラットフォームです。

> 私が韓国語で送ると、相手には自然な日本語で届く。
> 상대가 일본어로 보내면, 나에게는 자연스러운 한국어로。

## コア機能

### 1. AIリアルタイム翻訳チャット

メッセージを送ると、AIが**2秒以内に自動翻訳**します。翻訳文（大きく）と原文（小さく）が同時に表示され、語学学習にも役立ちます。

```
🇯🇵 私: "東京旅行はどうでしたか？"
          ↓ AI翻訳 (< 2秒)
🇰🇷 相手の画面: "도쿄 여행 어땠어요？"
                  東京旅行はどうでしたか？ ← 原文(小さく)
```

### 2. 趣味ベースのマッチング

15の興味タグ（旅行、グルメ、カフェ、音楽、ファッションなど）で趣味の合う相手を見つけます。ユーザーカードでプロフィールを確認し、すぐに会話を始められます。

### 3. バイリンガルUI

国籍選択と同時にUI全体が該当言語に切り替わります。韓国ユーザーは韓国語UI、日本ユーザーは日本語UIで自然に利用できます。

### 4. かんたんスタート

Googleログイン一回ですぐに開始。複雑な登録手続きなしに、プロフィール設定後すぐにチャットできます。
