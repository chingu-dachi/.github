친구다치란? チングダチとは？

한국과 일본 간 여행 수요가 폭발적으로 늘고 있지만, 언어 장벽이 가장 큰 허들입니다. 친구다치는 AI 실시간 번역으로 한국어와 일본어 사이의 벽을 허물어, 양국 사용자가 모국어로 자연스럽게 대화할 수 있는 채팅 기반 네트워킹 플랫폼입니다. 日本と韓国の間で旅行需要が爆発的に増加していますが、言語の壁が最大の障壁です。チングダチはAIリアルタイム翻訳で韓国語と日本語の壁を取り払い、両国のユーザーが母国語で自然に会話できるチャットベースのネットワーキングプラットフォームです。

핵심 기능 コア機能

🤖

AI 실시간 번역 AIリアルタイム翻訳

메시지를 보내면 2초 이내에 자동 번역. 번역문과 원문이 동시에 표시되어 언어 학습에도 도움됩니다. メッセージを送ると2秒以内に自動翻訳。翻訳文と原文が同時に表示され、語学学習にも役立ちます。

🎯

관심사 매칭 趣味マッチング

15개 관심사 태그로 취향이 맞는 상대를 찾아 바로 대화를 시작할 수 있습니다. 15の興味タグで趣味の合う相手を見つけてすぐに会話を始められます。

🌏

이중 언어 UI バイリンガルUI

국적 선택 즉시 전체 UI가 해당 언어로 전환됩니다. 国籍選択と同時にUI全体が該当言語に切り替わります。

⚡

간편한 시작 かんたんスタート

Google 로그인 한 번이면 프로필 설정 후 즉시 채팅 시작. Googleログイン一回でプロフィール設定後すぐにチャット開始。
